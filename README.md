# AWS EC2 Web Server Deployment

## Project Overview
This project demonstrates launching and configuring a web server on AWS EC2.

## What I Did
- Launched an EC2 instance on AWS
- Connected to the server using SSH
- Installed Apache web server
- Created and deployed a web page
- Configured security groups to allow HTTP access

## Result
Successfully hosted a live web page accessible via public IP.

## Technologies Used
- AWS EC2
- Apache (httpd)
- SSH
- Linux
